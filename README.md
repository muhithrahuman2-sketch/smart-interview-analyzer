# 🚀 Smart Prep Analyst

*Smart Prep Analyst* is an automated career and academic readiness platform designed to help students prepare for their future goals. It uses AI to analyze skills and provide insights for better preparation.

## ✨ Features
* *AI Interview Analysis*: Deeply analyzes your interview performance.
* *Career Readiness*: Helps in academic and career planning.
* *Resume Support*: Feedback on your resume and skills.

## 🛠️ Tech Stack
* *Language*: Python
* *Framework*: Streamlit
* *AI Models*: Google Gemini / OpenAI (as per your code)

## 🚀 How to Run
1. Clone the repository.
2. Install requirements: pip install -r requirements.txt
3. Run the app: streamlit run app.py
